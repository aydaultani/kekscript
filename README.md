# kekscript
Simple stack based language.

```
== INSTRUCTIONS ==
[0] PUSH | Push a value to the stack | PUSH 10
[1] POP | Pop the first value from the stack | POP
[2] STACK | Print the stack | STACK
[3] EXIT | Exit the REPL | EXIT
[4] ADD | Add the first two numbers in the stack | ADD
[5] SUB | Subtract the first two numbers in the stack | SUB
[6] MULT | Multiply the first two numbers in the stack | MULT
[7] DIV | Divide the first two numbers in the stack | DIV
```
